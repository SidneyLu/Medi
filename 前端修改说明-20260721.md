# Medi 前端修改说明

**备份时间：** 2026-07-21  
**备份目录：** `Medi/frontend-backup-ui-20260721-0401`（不含 `node_modules`、`.next`）  
**工作目录：** `Medi/frontend`  
**说明：** 本次主要为前端交互与文案调整；后端 PostgreSQL 报告能力沿用仓库已有实现。

---

## 1. 页面文本调整

统一打磨各页中文文案，并按产品要求去掉界面中的中文句号「。」（英文句号 `.` 保留）。

涉及页面包括但不限于：

| 页面 | 路径 | 主要改动 |
|------|------|----------|
| 登录 / 注册 | `(auth)/login`、`(auth)/register` | 文案与占位符 |
| 健康主页 | `(protected)/dashboard` | 导航与说明文案 |
| 循证问答 | `(protected)/chat` | 标题「大模型循证问答」、输入区文案 |
| 历史记录 | `(protected)/history` | 说明改为「汇总当前账户的健康咨询和体检报告记录（数据来自 PostgreSQL）」等 |
| 报告上传 / OCR / 解读结果 | `reports/**` | 步骤说明、提示语；结果页摘要展示时去掉中文句号 |
| 个人画像 | `(protected)/profile` | 说明与占位符 |

结果页文件：`src/app/(protected)/reports/[id]/result/page.tsx`  
辅助函数 `withoutChinesePeriods` 仅去除中文「。」，不影响英文句号与文件名中的 `.pdf`。

## 2. 聊天框回车键位

**行为：**

- `Enter`：发送消息  
- `Shift + Enter`：换行  
- 中文输入法组合输入过程中按 Enter 不发送（`isComposing` 判断）

**修改文件：**

- `src/app/(protected)/chat/page.tsx`（循证问答）  
- `src/app/(protected)/dashboard/page.tsx`（健康主页内嵌对话）

---

## 3. 退出登录；点击用户名 / 头像进入画像页

**修改文件：** `src/components/app-shell.tsx`

- 顶部栏用户名与头像区域改为可点击链接，跳转 `/profile`  
- 「退出登录」使用主题主按钮样式；退出时清理本地 token / React Query 缓存，并跳转登录页  

---

## 4. 报告解读三步骤可回退跳转

**修改文件：** `src/components/report-steps.tsx`  
**样式：** `src/app/globals.css`（`.step` 改为 button、`.clickable`）

**行为：**

- 步骤做成按钮  
- **仅可点击已走过的前置步骤**（例如在第 3 步可回第 1、第 2 步）  
- 当前步与后置步点击不响应  
- 跳转关系：  
  - 步骤 1 → `/reports`  
  - 步骤 2 → `/reports/{id}/ocr`（需 `reportId`）  
  - 步骤 3 → `/reports/{id}/result`（需 `reportId`）

在上传页、OCR 页、结果页分别传入 `step` / `reportId`。

---

## 5. 在上传区显示报告文件

解决「选中/上传后看不清是否成功」的问题。

**修改文件：**

- `src/app/(protected)/reports/page.tsx`：选择文件后展示文件卡片（文件名、类型、大小、「预览 PDF」、移除）  
- `src/app/(protected)/reports/[id]/ocr/page.tsx`：核对页顶部展示「已上传」文件卡片  
- `src/app/globals.css`：新增 `.upload-file-card`、`.icon-clear`、`.save-hint` 等样式  

另：OCR 页「保存核对结果」改为本地草稿编辑 + 显式保存，成功提示「核对结果已保存」；该按钮只写回服务器，不进入解读页。「确认并生成解读」才会跳转结果页。

---

## 6. 健康画像出生日期年份范围

**修改文件：** `src/app/(protected)/profile/page.tsx`

- `<input type="date" min="1900-01-01" max="2026-12-31" />`  
- Zod 校验：出生日期须在 `1900-01-01`～`2026-12-31`  

浏览器日期选择器年份范围与表单提交校验双重限制。
