# Medi 更改说明（相对上次提交 `a68ee33`）

上次已推送：强制登录中间件与 token cookie 同步。  
本次提交在此基础上，补齐**健康画像个性化**、**多轮会话记忆**、**历史删除**，以及相关前后端联调能力。

## 功能变更

### 1. 健康画像个性化（前后端）

- 后端新增 `format_profile_context`：把画像表单（称呼、年龄、性别、妊娠、体格、慢病、过敏、用药）整理成可读上下文，写入大模型提示词。
- 检索与生成继续支持 `use_profile`；有完整表单内容时不只用标签，还会带入全文画像摘要。
- 聊天页可勾选「使用我的健康画像」。

### 2. 多轮会话记忆（前后端）

- 请求体新增 `use_memory`（默认开启）。
- 开启时，将本会话近期历史（约 8 条）纳入检索扩展与回答生成，便于理解追问与指代。
- 聊天页可勾选「使用多轮会话记忆」。
- 未配置 API Key 时的占位回答改为「【本地联调自检】」，便于对照开关是否生效。

### 3. 历史记录管理

- 后端新增 `DELETE /api/v1/chat/conversations/{id}`，支持删除咨询会话。
- 历史页：咨询与报告条目均可右侧删除；报告文件名过长时中间省略显示。

### 4. 其它体验

- 新咨询默认标题改为「新咨询」。
- 仪表盘等文案/样式小幅调整，与历史列表布局对齐。
